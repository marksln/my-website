Bitwise Operators

  • & (AND) Returns a 1 in each bit position where both operands have a 1, otherwise 0.
  • | (OR) Returns a 1 in each bit position where either operand has a 1, otherwise 0.
  • ^ (XOR) Returns a 1 in each bit position where the operands are different, otherwise 0.
  • ~ (NOT) Inverts all bits of a single operand (1 becomes 0, and 0 becomes 1).
